一、系统配置

1.CentOS安装中文语言包 
vi /etc/sysconfig/i18n

LANG="zh_CN.UTF-8"
LC_ALL="zh_CN.UTF-8"
安装中文字体 
yum install fonts-chinese.noarch
重新登录后输入locale命令和date命令可以查看。

2.修改主机名 临时修改，用 
hostname=你的主机名
永久修改，用 
vi /etc/sysconfig/network
找到HOSTNAME所在行，改为（没有这一行则直接添加） 
HOSTNAME=你的主机名

3.修改时间 
date -s "2012-08-27 02:26:45"

4.目前每次开机时需要做的操作 
iptables -F
postfix start
date -s "2012-08-27 02:26:45"

二、安装、升级LAMP

安装apache、php和mysql 
yum -y install httpd php mysql mysql-server php-mysql
安装httpd的扩展组件 
yum -y install httpd-manual mod_ssl mod_perl mod_auth_mysql
安装php的扩展组件 
yum -y install php-mcrypt php-gd php-xml php-mbstring php-ldap php-pear php-xmlrpc php-imap
安装mysql 的扩展组件 
yum -y install mysql-connector-odbc mysql-devel libdbi-dbd-mysql
检查php的版本号，是不是低于5.2（通过#rpm -q php）。如果低于5.2，需要升级以支持后面将要安装的phpmyadmin。
由于CentOS默认的源中没有5.2版本的，我们需要从其他源上下载： 
vi /etc/yum.repos.d/CentOS-Base.repo
增加下面信息 
[utterramblings] 
name=Jason's Utter Ramblings Repo 
baseurl=http://www.jasonlitka.com/media/EL$releasever/$basearch/ 
enabled=1 
gpgcheck=1 
gpgkey=http://www.jasonlitka.com/media/RPM-GPG-KEY-jlitka
然后升级php: 
yum update php -y

三、配置LAMP

1.检测httpd和php是否已安装好 
chkconfig httpd on
service httpd start
启动httpd后，打开http://hostname，就能看到httpd是否正常运行。
新建一个文本，键入如下代码 命名为test.php，放入httpd的文档根目录下（/var/www/html），然后在本地电脑中看能否浏览。
曾经出现的一个httpd不能正常启动的原因是被事先安装的防火墙屏蔽了。用 
iptables -F
清除防火墙的所有过滤规则。

2.配置mysql 
chkconfig mysqld on
mysql默认的字符集可能不支持中文。可以从这里检查： 
mysql -u root -p
mysql> show variables like 'character%';
如果看到一堆latin1，就不对。应该都是utf8。
先停止mysql服务： 
service mysqld stop
修改配置文件 
vi /etc/my.cnf
找到客户端配置[client] 在下面添加 
default-character-set=utf8
找到[mysqld] 添加 
default-character-set=utf8
重新启动mysql: 
service mysqld start
重新检查一下，涉及字符的都是utf8就对了。

3. MySQL忘记密码怎么办？
首先停掉mysqld服务： 
service mysqld stop
如果 MySQL 正在运行，首先杀之： 
killall -TERM mysqld
启动 MySQL ： 
safe_mysqld --skip-grant-tables &
就可以不需要密码就进入 MySQL 了。
然后就是 
>use mysql 
>update user set password=password("你的新密码") where user="root"; 
>flush privileges;
重新杀 MySQL ，用正常方法启动。

四、安装phpMyAdmin

将下载的phpMyAdmin程序包（后缀名是tar.gz或者zip）上传到/var/www/html目录下。 
cd /var/www/html
用 
tar -xzvf 文件名
或者 
unzip 文件名
将程序包解压缩，得到一个目录。将目录重命名： 
mv 原目录名 phpmyadmin
在本地电脑浏览器上输入http//hostname/phpmyadmin，即可使用。
用root帐号登录，登录后为了安全一般要在数据库mysql的表user中，将user为空的记录删掉。并修改root用户的密码。另外创建一个新的数据库供emailmarketer使用。

五、安装配置emailmarketer

安装：解压后有几个目录或者文件要改写权限。 一律用 
chmod -R 777 目录或文件
即可。

另外email marketer经常遇到不能输入中文问题。
这时首先要确保一.1和三.3配置正确，然后删除原来的email marketer目录（#rm -rf emailmarketer），重新解压缩安装即可。
